# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct,
version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be welcoming, patient, and kind.
- Assume good faith. People come from different backgrounds, time
  zones, and skill levels.
- Disagree on the substance, not the person. Critique code and
  ideas, not contributors.
- No harassment, discrimination, or personal attacks.
- This is a hobby project that others are trying to build on top
  of — treat that with respect both ways.

## Scope

This Code of Conduct applies in all project spaces (issues, pull
requests, commit comments, discussions) and when an individual is
officially representing the project in public spaces.

## Reporting

To report unacceptable behavior, email **jaspercurry@gmail.com**.
All reports will be reviewed and investigated promptly and fairly.
The project owner is obligated to respect the privacy and security
of the reporter.

## Enforcement

The project owner is responsible for clarifying and enforcing this
Code of Conduct. Consequences range from a private warning to a
permanent ban, depending on severity and pattern. See the full
[Contributor Covenant 2.1 enforcement
guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
for the framework used.
