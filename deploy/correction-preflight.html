Room correction

The next page measures your room using this device's microphone. Browsers only allow that on HTTPS, so this one JTS screen has to switch from normal HTTP to HTTPS.

You may see "Your connection is not private." That is expected here. The speaker uses a local certificate because it lives on your home network, not on the public internet.

  1. Tap OK, proceed below.
  2. On the browser warning, open Show Details or Advanced, then choose the option to continue to this website.

To silence the warning permanently on this device, install and trust the JTS root certificate from the room-correction page.
